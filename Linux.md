## What is Operating System ? 
- An Operating system is a core software that manages the hardware and software on a computer. It acts as a bridge between user and machine.

### Types of Operating systems 
   - Desktop Operating system - Windows OS, Mac OS, Linux such as Ubuntu, fedora.
   - Server Operating system - Windows Server, Red Hat Enterprise Linux (RHEL)
   - Mobile Operating system - Andriod, iOS
   - Embedded Operating system - Runs on small devices with limited hardware like microwaves, routers, smart TVs.
   - Real Time Operating system - Used where immediate response is required like robots, medical devices, aircraft systems.

## What is Linux ?
- Linux is Open source operating system based on Unix Architecture.
   It manages hardware runs applications and provides secure and stable environment for computing.

   -  Open‑source — anyone can modify or distribute it
   - Secure — strong permissions and user control
   - Stable — rarely crashes, ideal for servers
   - Multi‑user — many users can work at once
   - Multitasking — runs many processes simultaneously
   - Portable — runs on phones, PCs, servers, routers, IoT

##  About Unix 
- Unix was developed to create a simple, portable, multi user, multi tasking operating system that could run on different hardware and support research.

- Unix is a proprietary, commercial operating system developed in the 1970s by AT&T Bell Labs.
   - Used in: large enterprises, banks, telecom
   - Examples: AIX, Solaris, HP‑UX
   - Very stable but expensive and closed‑source
