# Compress and Uncompress Commands
- compress reduces file size using .Z format.
- uncompress restores the original file.
- These are older UNIX commands, still available on many systems for compatibility.
- Modern Linux uses: gzip / gunzip, zip / unzip, tar

## tar - Archive files
- tar = “tape archive”
- It combines many files into one archive file (usually .tar, .tar.gz, .tgz).
- Important: tar itself does NOT compress.
- Compression happens only when used with -z (gzip) or -j (bzip2).
- Basic Syntax: ***tar [options] archive-name.tar files...***
- Most Important Options
  
      - | **Option** | **Meaning**                                     |
        |-----------|--------------------------------------------------|
        | **-c**    | Create archive                                   |
        | **-x**    | Extract archive                                  |
        | **-v**    | Verbose (show files)                             |
        | **-f**    | File name (must be last option before filename)  |
        | **-z**    | Compress using gzip                              |
        | **-j**    | Compress using bzip2                             |
        | **-t**    | List contents                                    |

- Create a tar archive - ***tar -cvf backup.tar folder/***
- Create a compressed tar.gz archive - ***tar -czvf backup.tar.gz folder/***
- Extract a tar.gz archive - ***tar -xzvf backup.tar.gz***
- List contents of a tar file - ***tar -tvf backup.tar***

## gzip — Compress files
- ***gzip*** compresses a single file and replaces it with .gz version.
- It is more modern and efficient than the old compress command.
- Basic Syntax -  ***gzip [options] file***
- Useful Options (from sources)
    - ***-k*** → keep original file (otherwise gzip deletes it)
    - ***-v*** → verbose output (show compression details)
    - ***-1 to -9*** → set compression level (1 = fast, 9 = best) 
- Compress a file - ***gzip file.txt*** - Creates file.txt.gz.
- Compress but keep original - ***gzip -k file.txt***
- Highest compression - ***gzip -9 file.txt***

## gunzip or gzip -d - Decompress .gz files
- gunzip restores .gz files back to original form.
- Basic Syntax - ***gunzip [options] file.gz***
- Useful Options (from sources)
    - **-k** → keep .gz file after decompressing 
    - **-c** → output to terminal (stdout) instead of creating a file
    - **-f** → force overwrite without asking
    - **-r** → recursive decompress in directories
    - **-l** → list compression details (ratio, size)
    - **-t** → test integrity (check if file is corrupted)
- Decompress a file- ***gunzip file.txt.gz***
- Keep original .gz - ***gunzip -k file.txt.gz***
- View content without extracting - ***gunzip -c file.txt.gz***
- Test if gzip file is valid - ***gunzip -t file.txt.gz***
