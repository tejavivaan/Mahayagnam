# Combining Files and Splitting Files
## Combining Files in Linux
- **cat — The simplest way to combine files**
  - cat joins files in the order you specify.
  - Combine two files into one - ***cat file1.txt file2.txt > combined.txt***
  - Append one file to another - ***cat file2.txt >> file1.txt***
- **paste — Combine files side‑by‑side**
  -  Useful when you want columns next to each other.
  -  Combine line‑by‑line horizontally
  -  ***paste file1.txt file2.txt***
- **join — Combine files based on a matching field**
  - Works like SQL JOIN (requires sorted files).
  - Join two files on the first column
  - ***join file1.txt file2.txt***
## Splitting Files in Linux
- **split — Split files by size or number of lines**
  - Creates multiple smaller files.
  - Split by number of lines
  - ***split -l 1000 bigfile.txt part_***
- **Split by size**
  - ***split -b 10M bigfile.txt chunk_***
  - Splits into 10 MB chunks.
- **Split into N equal parts**
  - ***split -n 4 bigfile.txt section_***
  - Creates 4 equal‑sized files.
