# Copying Files & Directories in Linux
- **cp stands for copy**.It is used to: Copy files, Copy directories.
## To copy a File
- Basic Syntax : ***cp [options] source destination***
- Copy a file (basic use)
  - ***cp file1.txt file2.txt***
  - Copies file1.txt
  - Creates a new file file2.txt
- Copy file into a directory
  - ***cp file1.txt /home/user/docs/***
  - Copies file into a folder
  - Keeps original name
- **- i** (interactive mode)
  - ***cp -i file1.txt file2.txt***
  - Asks before overwriting existing files
- **-f** (force copy)
  - ***cp -f file1.txt file2.txt***
  - Forces overwrite
  - No questions asked
- **-v** (verbose mode)
  - ***cp -v file1.txt file2.txt***
  - Shows what is being copied
- **-p** (preserve attributes)
  - ***cp -p file1.txt file2.txt***
  - Preserves: Permissions, Owner, Timestamp
- **-u** (update copy)
  - ***cp -u file1.txt file2.txt***
  - Copies only if source is newer or destination doesn’t exist
- **-a** (archive mode)
  - ***cp -a dir1 dir2***
  - Best for full backups
  - Preserves everything: permissions, timestamps, symbolic links, recursive copy
- **-n** (no overwrite)
  - ***cp -n file1.txt file2.txt***
  - Will NOT overwrite existing file
 
- Important cp options
  - | Option      | Meaning                      |
    | ----------- | ---------------------------- |
    | `-i`        | Ask before overwrite         |
    | `-f`        | Force overwrite              |
    | `-r` / `-R` | Copy directories             |
    | `-v`        | Show copy process            |
    | `-p`        | Preserve attributes          |
    | `-u`        | Copy only if newer           |
    | `-a`        | Full archive backup          |
    | `-n`        | No overwrite                 |
    | `--backup`  | Create backup of destination |

## To copy a directory
- we use the cp command with the -r (recursive) option.
- Basic Command to Copy a Directory.
  - ***cp -r source_directory destination_directory***
  - ***cp -r myfolder backup/*** - This copies myfolder into the backup directory. 
- Copy a Directory to Another Location.
  - ***cp -r /home/user/docs /home/user/Desktop/*** -  This copies the docs folder to the Desktop.
- Copy a Directory and Rename It.
  - ***cp -r project project_backup*** - This creates a new directory named project_backup.
- Copy Multiple Directories at Once.
  - ***cp -r dir1 dir2 dir3 /home/user/backup/*** - All three directories go into backup.
