# Linux Vs Windows Commands
- | **Linux Command** | **Windows Command** | **Purpose** |
|-------------------|---------------------|-------------|
| **ls** | **dir** | List files and directories |
| **cp** | **copy** | Copy files |
| **mv** | **move / rename** | Move or rename files |
| **rm** | **del / erase** | Delete files |
| **mkdir** | **mkdir** | Create directory |
| **rmdir** | **rmdir** | Remove empty directory |
| **pwd** | **cd** (alone) | Show current directory |
| **cd** | **cd** | Change directory |
| **cat** | **type** | Display file content |
| **clear** | **cls** | Clear terminal screen |
| **touch** | **type nul > file** | Create empty file |
| **find** | **dir /s** | Search for files |
| **grep** | **findstr** | Search text inside files |
| **man** | **help** | Show help/manual |
| **chmod** | **icacls** | Change permissions |
| **chown** | **takeown** | Change file owner |
| **ps** | **tasklist** | List running processes |
| **kill** | **taskkill** | Terminate processes |
| **ifconfig / ip** | **ipconfig** | Network configuration |
| **scp** | **copy /y** (not same) | Secure copy between systems |
| **ssh** | **ssh** | Remote login |
| **df -h** | **wmic logicaldisk** | Disk usage |
| **top / htop** | **taskmgr** | System monitoring |
