# Process, Job and Scheduling
##  Some Important Terms
- **Application** : An application is a program you use to do a task.
- **Script** : A script is a text file with commands written inside it.
- **Process** : A process is a running program.
- **Daemon** : A daemon is a special background process that runs all the time.
- **Threads** : A thread is a small part of a process that does one task.
- **Job** : A job is a task started from the terminal.
