# Process Management
- **What is a Process?**
  - A process is simply a program in execution.
  - Whenever you run something — a browser, a command, a script — Linux creates a process.

## Foreground & Background Processes (fg, bg)
### Foreground Process
- Runs in front of the terminal.
- You cannot type anything else until it finishes.
- Example: ***sleep 10***
### Background Process
- Runs behind the terminal.
- You can continue using the terminal.
#### How to send a process to background:
- Start directly in background: ***command &***
- Stop a running process using ***Ctrl + Z***
- Then send it to background: ***bg***
- How to bring a background process to foreground: ***fg***
- If multiple jobs: ***fg %2***
- View jobs: ***jobs***
