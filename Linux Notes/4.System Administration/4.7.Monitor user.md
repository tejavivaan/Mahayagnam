# MONITOR USER IN LINUX
- Monitoring users in Linux means tracking who is logged in, what they are doing,
  what processes they run, their resource usage, login history, and security‑related activities.


## What Does “Monitor User” Mean?
- Monitoring a user includes:
  - Tracking logged‑in users
  - Monitoring user activity
  - Checking user processes
  - Checking resource usage
  - Monitoring login history
  - Monitoring failed logins
  - Monitoring sudo usage

## Commands to See Logged‑In Users
- **who** — Who is logged in
  - Shows currently logged‑in users
  - Displays: username, terminal, login time, IP
- **w** — Who + What they are doing
  - Shows logged‑in users + their activity
- **users** — Only usernames
  - Very short output
- **last** — Login history
  - Shows all successful logins
- **lastb** — Failed login attempts
  - Reads /var/log/btmp
## Monitor User Activity in Real Time
- **who -u** — Logged‑in users + idle time
- **id username** — User identity details
  - Shows UID, GID, groups.
- **finger username**, If installed, shows:
  - Login time
  - Idle time
  - Home directory
  - Shell
## Monitor User Processes
- **ps -u username** - Shows all processes of a user.
- **top -u username** - Real‑time process monitoring for a specific user.
- **htop** (if installed) - Interactive, colorful process viewer.

## Monitor User Resource Usage
- CPU usage by user - **ps -u username -o pid,ppid,%cpu,%mem,cmd**
- Memory usage by user - **ps -u username --sort=-%mem**
- Disk usage by user home directory - **du -sh /home/username**
- Network usage (if tools installed)
-  **sudo nethogs** - Shows per‑user network usage.
-  **sudo iftop** - Shows real‑time bandwidth usage.

## Monitor User Security & Authentication
- Check failed logins - **sudo lastb**
- Check authentication logs
  - Location varies by distro:
    - RHEL/CentOS: ***sudo cat /var/log/secure***
    - Ubuntu/Debian: ***sudo cat /var/log/auth.log***
  - Look for: Failed password, Invalid user, sudo attempts

## Monitor Sudo Usage
- View sudo logs
  - ***sudo cat /var/log/secure | grep sudo*** or ***sudo cat /var/log/auth.log | grep sudo***
- Check who used sudo - ***sudo grep -i "sudo" /var/log/auth.log***
## Monitor User Login Sessions
- **whoami** - Shows current user.
- **tty** - Shows terminal session.
- **lastlog** - Shows last login of all users.
## Monitor User Commands (History)
- View user’s command history - ***cat /home/username/.bash_history***
- Real‑time command monitoring (advanced)
- Using auditd - ***sudo ausearch -ua username***
- Using psacct - ***sudo sa***
## Monitor User File Activity
- Using inotifywait
- Monitor file changes: ***inotifywait -m /home/username***
- Track file access: ***auditctl -w /home/username -p war -k userwatch***
