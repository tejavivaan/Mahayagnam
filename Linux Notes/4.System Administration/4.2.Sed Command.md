# SED — Stream Editor (Complete Notes)
- **sed** stands for Stream EDitor.
- **sed** is a stream editor used to modify text in a pipeline without opening the file.
- It reads input line by line, applies the given commands, and prints the result.
- Basic Syntax: **sed [options] 'command' file**
- Important Commands
  - **s** → substitute
  - **g** → global
  - **d** → delete
  - **p** → print
  - **i** → insert
  - **a** → append
  - **c** → change line

- Important Options
  - **-n** → suppress output
  - **-i** → in-place edit
## Most Important sed Commands
### Substitution — s command
- Replace first occurrence in each line: ***sed 's/cat/dog/' file***
- Replace all occurrences (global): ***sed 's/cat/dog/g' file***
- Replace only on a specific line number: ***sed '3s/cat/dog/' file***
- Replace only in lines matching a pattern: ***sed '/animal/s/cat/dog/'***
  
### Deleting Lines — d command
- Delete a specific line: ***sed '3d' file***
- Delete a range of lines: ***sed '5,10d' file***
- Delete lines containing a pattern: ***sed '/error/d' file***

### Printing Lines — p command
- Print only lines matching a pattern: ***sed -n '/hello/p' file***
- Print a specific line: ***sed -n '5p' file***
- Print a range: ***sed -n '3,7p' file***

### Insert & Append Text
- Insert BEFORE a line — i: ***sed '3i This is inserted before line 3' file***
- Append AFTER a line — a: ***sed '3a This is appended after line 3' file***

### Change (Replace Entire Line) — c
- Replace Entire Line: ***sed '4c This is the new line 4' file***
### Replace only nth occurrence
- ***sed 's/apple/orange/2' file***
- Replaces 2nd occurrence in each line.

## Useful Options
### -n (Suppress automatic printing)
- sed -n 's/cat/dog/p' file
- Prints only modified lines.

### -i (Edit file in-place)
- sed -i 's/cat/dog/g' file
- This modifies the file permanently.

## Examples
- Replace text only between two patterns: ***sed '/start/,/end/s/old/new/g' file***
- Remove blank lines: ***sed '/^$/d' file***
- Remove leading/trailing spaces
  - Leading: ***sed 's/^[ \t]*//' file***
  - Trailing: ***sed 's/[ \t]*$//' file***
