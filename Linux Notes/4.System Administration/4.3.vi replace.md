# vi uses a sed‑style substitute command
- The vi replace command is basically the same syntax as the sed substitute command.
- Both use: ***s/pattern/replacement/flags***
## sed substitute command
- sed is a stream editor — it edits text non‑interactively.
- Example:
  - ***sed 's/old/new/g' file.txt***
  - This prints the modified output to the terminal.
  - To save changes:
  - ***sed -i 's/old/new/g' file.txt***
## vi substitute command
- Inside vi, you use the same syntax, but with a colon command:
  - ***:%s/old/new/g***
- Differences:
  - vi edits inside the editor
  - sed edits from the command line
  - vi uses ranges like %, 1,10, ., $
  - sed uses line numbers or patterns

### Replace in the CURRENT LINE
- ***:s/old/new***
- Replaces first occurrence of “old” in the current line.
### Replace ALL occurrences in the CURRENT LINE
- ***:s/old/new/g***
- Adds g (global) to replace all matches in that line.

### Replace in the ENTIRE FILE (most used)
- ***:%s/old/new/g***
- Breakdown:
  - % → entire file
  - s → substitute
  - g → replace all occurrences
- This is the most common replace command in vi.

### Replace with CONFIRMATION (interactive)
- ***:%s/old/new/gc***
- c → confirm each replacement
- vi will ask:
  - y → yes
  - n → no
  - a → replace all
  - q → quit
  - l → replace and stop
- Very useful when you want to avoid mistakes.

### Replace FIRST occurrence in entire file
- ***:%s/old/new***
- No g → only first match in each line.

### Replace in a SPECIFIC RANGE of lines
- Replace between line 10 and 20:
  - ***:10,20s/old/new/g***
- Replace from current line to end:
  - ***:.,$s/old/new/g***
- Replace from top to current line:
  - ***:1,.s/old/new/g***
### Case‑Insensitive Replace
- ***:%s/old/new/gi***
- i → ignore case
- Matches OLD, Old, oLd, old, etc.

### Replace WHOLE WORDS only
- ***:%s/\<old\>/new/g***
  - **\<** → word start
  - **\>** → word end
- Prevents replacing inside other words
- Example: replacing “cat” won’t change “concatenate”.

### Replace using REGEX (advanced)
- Replace digits with nothing:
  - ***:%s/[0-9]//g***
- Replace multiple spaces with one:
  - ***:%s/ \+/ /g***
- Replace tabs with spaces:
  - ***:%s/\t/    /g***
