# top & htop Command
## top 
- A built‑in Linux command that shows real‑time system performance:
  - CPU usage
  - Memory usage
  - Running processes
  - Load average
  - Tasks
  - System uptime
- It is text‑based, keyboard‑driven, and available on all Linux systems by default.

### Why Do We Use These Commands?
- **System Monitoring** : To check system health in real time.
-  **Troubleshooting** : To identify high CPU, memory, or load issues.
-  **Process Management** : To kill, renice, or inspect processes.
-  **Performance Analysis** : To understand resource consumption patterns.

### Understanding top Command 
- How to Run : ***top***
- top Output Sections
- Header Section Shows system‑wide information:
  - uptime — how long system is running
  - load average — system load (1, 5, 15 min)
  - tasks — total, running, sleeping, stopped, zombie
  - CPU usage — user, system, idle, iowait
  - memory usage — total, used, free, buffers
  - swap usage — total, used, free, cached
- Process Table, Columns include:
  - | Column | Meaning |
    | --- | --- |
    | PID | Process ID |
    | USER | Owner of process |
    | PR | Priority |
    | NI | Nice value |
    | VIRT | Virtual memory |
    | RES | Physical memory |
    | SHR | Shared memory |
    | S | State (R,S,D,T,Z) |
    | %CPU | CPU usage |
    | %MEM | Memory usage |
    | TIME+ | Total CPU time |
    | COMMAND | Process name |
### Important Keyboard Shortcuts in top
- Sorting Options
  - ***P*** → Sort by CPU
  - ***M*** → Sort by Memory
  - ***T*** → Sort by Time
- Killing Processes
  - Press ***k*** → enter PID → send signal (default 15)
- Changing Refresh Rate
  - Press ***d*** → set delay in seconds
- Search Process
  - Press ***/*** → type process name
- Quit
  - Press ***q***

## htop
- An advanced, colorful, interactive version of top.
- Easier to read, Mouse support, Scrollable, Tree view, Kill processes easily
- Not installed by default → must install manually.
### Understanding htop Command 
- **Installation**
  - sudo apt install htop     # Ubuntu/Debian
  - sudo yum install htop     # RHEL/CentOS
  - sudo dnf install htop     # Fedora
- **Features of htop**
  - **Colorful UI** : CPU, memory, swap bars with colors.
  - **Mouse Support** : Click to kill, sort, scroll.
  - **Tree View** : Shows parent‑child process hierarchy.
  - **Scrollable Process List** : Unlike top, you can scroll up/down.
  - **Easy Process Management** F9 → kill, F7/F8 → renice

- **htop Interface Sections**
  - **Meters Section** : CPU bars, Memory bar, Swap bar, Load average, Uptime
  - **Process Table** : Columns similar to top but more readable.
### Useful Keyboard Shortcuts in htop
- | Key | Action |
  | --- | --- |
  | F2 | Setup menu |
  | F3 | Search |
  | F4 | Filter |
  | F5 | Tree view |
  | F6 | Sort by column |
  | F9 | Kill process |
  | F10 | Quit |
## top vs htop
- | Feature | top | htop |
  | --- | --- | --- |
  | Default availability | Yes | No |
  | Colorful UI | No | Yes |
  | Mouse support | No | Yes |
  | Scrollable | No | Yes |
  | Tree view | No | Yes |
  | Easy process kill | No | Yes |
  | Customization | Limited | High |
