# Switch User & Sudo Access in Linux

## Switch User (su)
- Switching user means changing your current identity to another user without logging out.
- **su = Substitute User or Switch User**
- Switch to Another User
  - ***su username***
  - You will be asked for that user’s password.

- Switch to Another User With Their Environment
  - ***su - username***
  - The ``-`` loads:
    - User’s PATH
    - User’s shell
    - User’s environment variables
    - User’s home directory
  - This is the correct way to switch users.

- Switch to Root User
  - ***su -***
  - You must know the root password.

- Run a Single Command as Another User
  - ***su - username -c "command"***
  - Example: ***su - john -c "ls /home/john"***
- Exit Back to Previous User
  - ***exit***
- Difference Between ``su`` and ``su -``
  - | Command | Meaning |
    | --- | --- |
    | ``su`` ``user`` | Switch user **without** loading environment |
    | ``su ``- ``user`` | Switch user **with full login environment** |
  - Always prefer: ``su - username``
### Why ``su`` Is Not Preferred in Modern Linux
  - Because:
  - It requires knowing the other user’s password
  - It requires knowing the root password
  - It is less secure
  - It cannot restrict permissions
  - This is why sudo is used instead.
 
## Sudo
- **sudo = Superuser Do**
- It allows a normal user to run admin/root commands without knowing the root password.
- Sudo is: Secure, Logged, Audited, Permission‑controlled
### How Sudo Works
- When a user runs: ***sudo command***
- Linux checks:
  - Is the user in the sudoers file?
  - If yes → allow
  - If no → deny

### Give Sudo Access to a User
- Add User to Sudo Group (Ubuntu/Debian)
  - ***sudo usermod -aG sudo username***
- Add User to Wheel Group (RHEL/CentOS/Fedora)
  - ***sudo usermod -aG wheel username***
-  Verify Sudo Access
  - ***groups username*** or ***sudo -l***

### Using Sudo
- Run a Command as Root
  - ***sudo command***
-  Run a Command as Another User
  -  ***sudo -u username command***
- Run a Root Shell
  - ***sudo -i*** or ***sudo su -***
