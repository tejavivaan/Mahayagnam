# ps Command
## What is ps? 
- The ps command shows the current running processes on your Linux system.
- Think of it as a snapshot of all processes at the moment you run the command.

### Why ps is Important?
- Because it helps you:
  - See which programs are running
  - Find process IDs (PID)
  - Check CPU/memory usage
  - Troubleshoot system issues
  - Identify hung or zombie processes
  - Kill unwanted processes

-  Basic Syntax  : ***ps [options]***
### Most Common ps Options 
- Below are the most important options you will use daily.
- Standard Options (Most Used)
- ***ps*** : Shows processes of the current shell only.
- ***ps -e*** : Shows all processes in the system.
- ***ps -f*** : Full format listing (more details).
- ***ps -ef*** : Shows all processes in full format.
- ***ps -aux*** : Shows all processes with CPU/memory usage.BSD-style output.
### Understanding ps -ef Output Columns
- When you run ps -ef, you see columns like:
  - | Column | Meaning |
    | --- | --- |
    | **[UID](ca://s?q=Explain_UID_in_ps_output)** | User who owns the process |
    | **[PID](ca://s?q=Explain_PID_in_ps_output)** | Process ID |
    | **[PPID](ca://s?q=Explain_PPID_in_ps_output)** | Parent Process ID |
    | **[C](ca://s?q=Explain_C_in_ps_output)** | CPU usage |
    | **[STIME](ca://s?q=Explain_STIME_in_ps_output)** | Start time |
    | **[TTY](ca://s?q=Explain_TTY_in_ps_output)** | Terminal associated |
    | **[TIME](ca://s?q=Explain_TIME_in_ps_output)** | CPU time consumed |
    | **[CMD](ca://s?q=Explain_CMD_in_ps_output)** | Command used to start process |

### Filtering Processes
- Search for a specific process : ***ps -ef | grep sshd***
- Search by command name : ***ps -C sshd***
- Search by PID : ***ps -p 1234***
- Sort processes by memory : ***ps aux --sort=-%mem***
- Sort processes by CPU : ***ps aux --sort=-%cpu***
- Custom output format like show PID, user, command : ***ps -eo pid,user,cmd***
- Show threads of a process : ***ps -T -p 1234***
- Show processes of a specific user : ***ps -u root***
- Long format listing : ***ps -l***
