# Linux File Editor
- A text editor is a program which enables you to create and manipilate data(text) in a Linux file.
- There are several standard text editors available on most Linux system.
  -  **vi** - Visual editor
  -  **ed** - Standard line editor
  -  **ex**  -  Extended line editor 
  -  **emacs** - A full screen editor
  -  **pico** - Beginner's editor
  -  **vim** - Advance version of vi
- Our editor = vi (available in almost every Linux distribution)

## vi Editor 
- vi is the default text editor available on every Linux/Unix system.
- It is modal, meaning it works in different modes for typing, navigating, and executing commands.
- Vim (Vi IMproved) is the enhanced version installed on most systems today.

### Command Mode (default mode)
- This is where you navigate, delete, copy, paste, and run commands.
- You enter command mode by pressing: **Esc**
- In this mode:
  - **dd** -  deletes a line
  - **yy** -  copies a line
  - **p**  -  pastes
  - **x**  -  deletes a character
  - **u**  - Undo
  - **r**  - replace a word
  - **/word** -  searches

### Insert Mode (typing mode)
- This is where you type text normally.
- Enter insert mode using:
  - **i** → insert before cursor
  - **a** → append after cursor
  - **o** → open a new line below
- Exit insert mode: **Esc**

### Command‑Line Mode (save, quit, replace)
- Enter this mode by typing: **:**
- This mode is for file operations and advanced commands:
  - **:w** → save
  - **:q** → quit
  - **:wq** → save & quit
  - **:q!** → quit without saving


## vim Editor 
- vim stands for Vi Improved. It is an enhanced version of vi with many additional features.
### Opening Files:
  - | Command              | Purpose             |
    | ------------------   | ------------------- |
    | **vim filename**     | Open file           |
    | **vim file1 file2**  | Open multiple files |
    | **vim**              | Open empty editor   |
    | **vim +10 file.txt** | Open at line 10     |
    | **vim + file.txt**   | Open at last line   |

### Insert Mode, Used for typing text
  - | Command | Purpose                     |
    | ------- | --------------------------- |
    | **i**     | Insert before cursor        |
    | **I**     | Insert at beginning of line |
    | **a**     | Insert after cursor         |
    | **A**     | Insert at end of line       |
    | **o**     | New line below              |
    | **O**     | New line above              |

### Saving and Exiting
  - | Command | Purpose                   |
    | ------- | ------------------------- |
    | **:w**    | Save file                 |
    | **:**     | Quit                      |
    | **:wq**   | Save and quit             |
    | **ZZ**    | Save and quit             |
    | **:q!**   | Quit without saving       |
    | **:wq!**  | Force save and quit       |
    | **:x**    | Save if modified and quit |

### Navigation Commands
  - | Command    | Purpose           |
    | ---------- | ----------------- |
    | **h**        | Move left         |
    | **l**        | Move right        |
    | **j**        | Move down         |
    | **k**        | Move up           |
    | **0**        | Beginning of line |
    | **$**        | End of line       |
    | **gg**       | First line        |
    | **G**        | Last line         |
    | **10G**      | Go to line 10     |
    | **Ctrl + f** | Page down         |
    | **Ctrl + b** | Page up           |

### Delete Commands
  - | Command | Purpose                   |
    | ------- | ------------------------- |
    | **x**     | Delete character          |
    | **dd**   | Delete current line       |
    | **5dd**   | Delete 5 lines            |
    | **D**     | Delete from cursor to end |
    | **dw**    | Delete word               |
    | **d$**    | Delete to end of line     |

### Copy and Paste
  - | Command | Purpose           |
    | ------- | ----------------- |
    | **yy**    | Copy current line |
    | **5yy**   | Copy 5 lines      |
    | **yw**    | Copy word         |
    | **p**     | Paste below       |
    | **P**     | Paste above       |

### Undo and Redo
  - | Command  | Purpose |
    | -------- | ------- |
    | **u**      | Undo    |
    | **Ctrl+r** | Redo    |

### Search Commands
  - | Command | Purpose         |
    | ------- | --------------- |
    | **/word** | Search forward  |
    | **?word** | Search backward |
    | **n**     | Next match      |
    | **N**     | Previous match  |
  
### Replace Commands
  - | Command         | Purpose                 |
    | --------------- | ----------------------- |
    | **r**             | Replace one character   |
    | **R**             | Replace continuously    |
    | **:%s/old/new/g** | Replace all occurrences |
    | **:s/old/new/g**  | Replace in current line |
